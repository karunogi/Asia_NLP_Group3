# 또, 오셨네요. RE,COMMA
RE,COMMA는 사용자의 질문을 NLP를 통해 분석하여, 사용자 맞춤형 콘텐츠를 추천해주는 대화형 콘텐츠 추천 챗봇입니다.

추천을 뜻하는 Recommendation과 쉼표라는 의미의 Comma를 합쳐 휴식을 추천해준다는 의미를 담고있습니다.
